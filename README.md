# resource-webs-ng28

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存放，不针对任何特定域名或网站，仅作为页面内容的整理与展示用途。

仓库结构简单，适合存放小型网页、演示页面、临时页面或静态内容集合。

## 目录说明

```
resource-webs-ng28/
├── index.html          # 仓库入口页面（如有）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下每个 HTML 文件为一个独立页面，彼此不依赖。
- `assets/` 目录存放页面可能引用的公共资源，按类型归类。

## 页面归档说明

- 所有页面均为静态 HTML，可在浏览器中直接打开运行。
- 页面内容与样式独立，不依赖外部在线服务。
- 归档目的为保存与展示，不保证长期更新或维护。

## 维护说明

- 本仓库不定期更新，新增或修改页面。
- 欢迎通过 Issue 提交建议或反馈页面问题。
- 如需添加新页面，请将 HTML 文件放入 `pages/` 目录，并更新相应资源引用。

## 许可

本仓库内容仅供学习与参考使用，未明确标注许可协议的页面默认保留所有权利。
